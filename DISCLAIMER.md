# Disclaimer

Cryptoxis is provided without warranty.

It uses a one-time-pad-style XOR approach. This is only secure if the generated key is truly random, kept secret, the same size as the data, and never reused.

If the key is lost, the encrypted file cannot be recovered.

Cryptoxis does not provide password-based encryption, authentication, tamper detection, secure key storage, or secure key exchange.

Use Cryptoxis at your own risk and review whether it fits your security needs before using it for sensitive data.
